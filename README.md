# Веб-ларек

---

Учебный проект, выполненный в рамках обучения
и доработанный мной для портфолио.

## О проекте
Веб-ларек — это веб-приложение для онлайн-заказа товаров.  
Реализованы каталоги товаров, корзина и оформление заказа с проверкой формы.

## Моя роль и вклад
В проекте я отвечала за:

- разработку и типизацию моделей данных (товары, заказы, корзина)
- работу с DOM и событиями через кастомный EventEmitter
- создание и поддержку модальных окон (товары, корзина, оформление заказа)
- реализацию логики корзины и форм заказа с валидацией
- структурирование кода по слоям MVP
- настройку сборки проекта через Webpack

## Используемые технологии
- TypeScript
- HTML / SCSS
- Webpack
- Модульная архитектура (MVP)
- Git/GitHub

## Чему я научилась
- работа с TypeScript и типами данных
- построение модульного архитектурного кода
